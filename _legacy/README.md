# Mysterious Personal Website

A dark, mysterious personal website where the cursor acts as a spotlight/magnifying glass, revealing hidden secrets and keys scattered across the page.

## Features

- **Dark Theme**: Completely blacked out background
- **Cursor Spotlight**: The cursor reveals hidden content like a magnifying glass
- **Hidden Secrets**: Multiple mysterious elements (keys, symbols) hidden throughout the page
- **Interactive Mystery**: Explore by moving your cursor to discover hidden content
- **Smooth Animations**: Elegant transitions and glow effects

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Technologies

- React 18
- Vite
- HTML/CSS/JavaScript

## How It Works

Move your cursor around the page to reveal hidden secrets. The cursor creates a spotlight effect that illuminates hidden keys and mysterious elements scattered across the dark canvas. Each revealed secret displays a message, creating an interactive mystery hunt experience.
